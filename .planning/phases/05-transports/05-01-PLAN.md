From RESEARCH.md Example 3 + CONTEXT.md D-08/D-10 + Pitfall 10 — TransportParamParser shape:
```swift
public enum TransportParamParser {
    public enum ParserError: Error, LocalizedError, Equatable {
        case wsMissingPath
        case httpMissingPath
        case httpUpgradeMissingPath
        case unsupportedType(String)
    }
    public static func parse(query: [String: String]) throws -> TransportConfig
}
```
Type dispatch rules:
- `type` absent / "tcp" / "raw" / "" → `.tcp`
- `type=ws` requires `path` (else throws wsMissingPath); `host` defaults to "" (caller substitutes SNI)
- `type=grpc` reads `serviceName` — default "TunService"
- `type=http` or `type=h2` requires `path` (else throws httpMissingPath)
- `type=httpupgrade` requires `path` (else throws httpUpgradeMissingPath); `host` defaults to ""
- other → throws `.unsupportedType(typeRaw)`
