- NWConnection(host: NWEndpoint.Host(host), port: NWEndpoint.Port(rawValue: UInt16(port))!, using: .tcp)

Swift Concurrency:
- ContinuousClock().now / clock.now - start = Duration; .components.seconds + .components.attoseconds для ms.
- CheckedContinuation с single-resume invariant — multiple state callbacks возможны (.ready then .cancelled), нужен LockedBool helper (RESEARCH Example 1).
- AsyncStream { continuation in Task { ... } continuation.onTermination = { _ in task.cancel() } } — для cancellation propagation (RESEARCH Example 2).
- TaskGroup parallel; внутри child task — sequential 3 probes с try? await Task.sleep(for: .milliseconds(50)) между ними.
